## Usage

These skills are used with [Claude Code](https://claude.ai/code) to provide context-aware assistance.

Skills are stored in `~/.claude/skills/` and automatically loaded when relevant.

## Installation

To use these skills, symlink or copy to your Claude Code skills directory:

```bash
# Clone this repo
git clone git@github.com:enisbudancamanak/claude-skills.git ~/.claude/skills
```
