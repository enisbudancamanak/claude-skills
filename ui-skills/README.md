# UI Skills

Opinionated constraints for building better interfaces with agents. This skill enforces strict guidelines for Tailwind CSS usage, animation standards (motion/react), accessible component primitives, and design consistency.

## Structure

-   `SKILL.md` - Main skill instructions containing the design and tech constraints.

## Usage

This skill is automatically discovered by Claude when requesting UI code, design implementations, or frontend tasks. It acts as a linter/guide for the agent to ensure high-quality, modern frontend code.
